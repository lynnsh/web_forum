##### web_forum
## Web Forum Project

_Tools: C#, ASP .NET MVC, MS SQL_

Description:
A web application that functions as a web forum and allows users to:
- read and post threads, 
- add and search by tags,
- read and answer to comments,
- upvote and downvote the thread.
